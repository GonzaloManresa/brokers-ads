# 🏢 TimeTrack — Sistema de Gestión Interna Brokers Ads

Aplicación web desarrollada como proyecto de tesis para gestión 
interna de una empresa de publicidad digital.

🔗 **[Ver aplicación en vivo](https://brokers-ads.vercel.app)**

## ✨ Funcionalidades
- Sistema de autenticación de usuarios
- Gestión interna de operaciones
- Modo oscuro / claro con persistencia en localStorage
- Interfaz responsive

## 🛠️ Stack tecnológico
- **Frontend:** HTML, CSS, JavaScript vanilla
- **Base de datos:** Supabase (PostgreSQL en la nube)
- **Deploy:** Vercel (producción)

## 📌 Contexto
Desarrollado como proyecto de tesis en la Universidad del Aconcagua, 
Mendoza. Aplicación funcional deployada en producción.
